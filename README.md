# 入退室記録アプリ (javaEE7 Practice)

## 概要

* 部屋に入ったり出たりした人や時間の記録をつける。

* 会社などの組織内にあるサーバにデプロイしてLAN内部で使う想定。

* どのユーザでログインしても記録表は同じ。組織ごとに記録表を分ける実装は今のところない。
